Unfinished Stories

Project Details

  • Category: Web design
  • Host: Daily Bruin
  • Project date: April 2020
  • Project URL: https://covidstories.dailybruin.com

Created and designed the title page of Unfinished Stories, a site where students around the world can submit their stories on how the pandemic has affected them. Used React to create animations and implemented frontend modules of a social media page with filtering and sorting functions of stories.
