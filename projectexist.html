Project Exist

Project information

  • Host: ACM Hackathon
  • Project date: Spring 2019
  • Project URL: https://github.com/janianovi/project-exist

Project Exist is a text-based college life game with a goal of maximizing the player's health, wealth, attractiveness, happiness, and intelligence stats. This game is inspired by the role playing game "A Dark Room". Personal contributions include creating the user interface design and implementing the life funtion modules within the game.
